Compatible Numbers
17 March 2023

For each element, the problem asks us to find a value which is a subset of its negation.
So for each subset of the 22 (~log2(4e6)) bits, we want to be able to check whether a subset of it exists in the given array. This can be done using DP.

Let's compute dp[mask] = any submask present in the array, if it exists. The submask can be equal to the mask if there exists a[i] = mask. Otherwise, it must be a submask of mask with a bit flipped off. This forms a DAG structure that we can simply iterate upwards on for a topological ordering.
